Ex12 Web Service

Accesses table `reviews` in database `iplab` at localhost:3306

Table specifications: name VARCHAR(20), age INT(3), rating INT(2), review VARCHAR(50)

Web Service methods:
  • recordReview(String name, int age, int rating, String review) returns String "OK" on success
  • showReview() returns String of all records in `reviews`
Access the Tester to check the service methods.
